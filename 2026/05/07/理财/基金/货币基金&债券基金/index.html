货币基金/债券基金

发表于2026-05-07|更新于2026-05-08|理财基金
|浏览量:

基础概念

专业名词

  1. 货币基金
  • 原理：只投银行存款、国债、高信用短债，几乎无风险、随取随用。
  • 收益低，不适合长期大额，跑不赢通胀
  1. 债券基金
  • 原理：债券本质就是国家、银行、企业向市场借钱，给你打欠条，约定到期还本、每年付利息
  • 收益来源：债券利息、债券市场利率价格波动（低买高卖)
  • 市场利息涨了 → 旧债券利息显得太低 → 没人愿意买 → 债券价格下跌
    市场利息跌了 → 旧债券利息更值钱 → 抢着买 → 债券价格上涨
  • 怎么看市场利息？
    • 10年期国债收益率：判断债基贵不贵、长期趋势（>3% 利好，<2.5% 警惕）https://www.chinamoney.com.cn/chinese/sddsintigy/
    • DR007/7 天逆回购：银行间借钱的7天成本，判断短期资金松紧、债基短期涨跌。https://www.chinamoney.com.cn/chinese/
      • DR007式市场钱多钱少，官方态度看7天逆回购率
      • DR007 > 逆回购利率：钱紧(>2%)，市场利率易涨，债基易跌。
      • DR007 < 逆回购利率：钱松(<1.3%)，市场利率易跌，债基易涨。
    • 5 年期 LPR：银行房贷，判断长期利率方向、降息 / 加息周期。https://www.chinamoney.com.cn/chinese/bklpr/
      • LPR 下调 = 降息周期 → 市场利率下行 → 债基上涨、股票偏强；
      • LPR 上调 = 加息周期 → 市场利率上行 → 债基下跌、股票偏弱；

需要做什么

  1. 10 年期国债收益率（管债券基金大趋势）
    • 作用：判断债基现在是便宜能买、还是贵要谨慎
    • ＞3.0%：利率高位 → 债基便宜，适合加仓拿长线
    • 2.5%～3.0%：中性正常 → 拿着不动就行
    • ＜2.5%：利率低位 → 债基偏贵，别重仓，只小额定投
  2. 央行 7 天逆回购利率（管政策大方向）
    • 作用：看央行放水还是收钱，定大基调
    • 数值很少变，几个月才动一次
    • 上调 = 收紧钱 → 对债基利空
    • 下调 = 放水宽松 → 对债基利好
    • 平时不动：货币政策平稳，不用瞎操心
  3. DR007（管短期资金松紧、货基 / 短债）
    • 作用：看当下市场钱多还是缺钱，天天波动
    • 1.5%～2.0%：正常区间，平稳
    • 持续＞2.0%：钱紧 → 货基收益会走高，债基容易震荡下跌
    • 持续＜1.3%：钱很松 → 货基收益变低，债基环境友好易涨
  4. 5 年期 LPR（管降息周期、楼市、大环境）
    • 作用：看长期是不是进入降息周期每月 20 号才更新，平时不用天天看
    • LPR 下调 = 进入降息周期 → 利好债基、股市
    • LPR 不动 = 维持现状
    • LPR 上调 = 加息预期 → 对债基不友好
