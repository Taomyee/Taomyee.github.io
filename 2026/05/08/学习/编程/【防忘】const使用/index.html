C语言 const 用法防忘指南

核心目的：记准 const 四种常用写法、避免踩坑，适合日常回顾、快速查阅，新手也能一眼看懂，杜绝记混！

一、基础作用（必记）

const 关键字的核心作用是修饰变量为“只读” ，编译阶段就会检查，防止代码中意外修改该变量的值，减少bug，尤其适合定义固定不变的数据（比如π、常量参数等）

⚠️ 重点：const 修饰的变量必须初始化，不能先声明、后赋值，否则编译报错。

二、四种常用写法

1. const 修饰普通变量（最基础）

2. const在*左边

管值，不管地址

3. const在*右边

管地址，不管值

4. 双重const

即管值，又管地址
