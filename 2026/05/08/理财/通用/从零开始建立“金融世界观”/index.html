从零开始建立“金融世界观”

发表于2026-05-08|更新于2026-05-09|理财通用
|浏览量:

https://www.bilibili.com/video/BV1Z4RHYKEcB?spm_id_from=333.788.videopod.sections&vd_source=35ed79a89e50ec8d5972f6695ef121ba

第一节：货币流动性

  1. 央行：货币发行和调控
  2. 商业银行：将货币传导到实体经济中，我理解为央行只有一个，商业银行作为中间媒介跟企业和居民产生联系
  3. 企业、居民：使用货币生产、投资、消费
  4. 商业银行会有一个资产表、一个负债表，我存了1w，就是银行有了1w，并且欠债1w（央行对商业银行也会有这两个表）
  5. 商业银行在外国人换汇后还会有外汇
  6. 商业银行可以把外汇向央行换成人民币
  7. 商业银行可以把持有的国债暂时“卖”给央行，并约定在7天后连本带利再“买“回来(央行逆回购)
    • 等于央行借钱给银行，短期向市场投放货币，注入流动性
    • 除了逆回购，还有常备借贷便利、中期借贷便利等，把钱借给银行

第二节：商业银行如何创造数倍存款（增加市场中的货币总量）

  1. 法定存款准备金率：央行规定，商业银行必须将一定比例的存款资金存入央行准备金账户，控制货币供应量和防范流动性风险
    • 我存了10w，如果存款准备金率是10%，那银行就要把1w存到央行，这样就只有9w用于放贷；而降低准备金率，银行就有更多的钱用于放贷了
  2. 而其实，小美贷款9w，只是账户数字变化而已，贷款9w，存款9w，这样银行就有了18w；也就是说，银行通过信用扩张，创造了货币；市场中总货币增多了，这笔因贷款而创造的货币，成为派生货币
  3. 存款准备金率是10%的情况下，10w能创造大概90w派生货币，100w/10w=10，为货币乘数（当前准备金率大概是6.6%，超额准备金率为1-2%）

第三节：黄金

  1. 价值共识：作为一般等价物，可以当作货币用，大家都认可
  2. 价值锚定：国家法定货币和黄金挂钩（7.32238g金本位~1英镑；美元和黄金挂钩，其他货币和美元挂钩的双本位）
  3. 通胀防御
  4. 风险闪避：闪避金融冲击
  5. 国家战略：战略储备值（下次货币变革），提升央行信用

第四节：利率

  1. 国债：把钱借给国家，无风险利率（1.5%）
  2. 银行根据无风险利率，以及信用情况+资产情况（抵押）做综合评估（2%）和运营成本（0.3）得出个人贷款3.8%
  3. 贷款市场报价利率LPR：由国内18家银行给出的平均报价
    • 一般就是LPR+银行加点（看运营情况）
    • 中小微企业可能还会有政策优惠-1%
  4. 同业拆借：上海18家商业银行间拆借利率Shibor1.7%
    • 那国债做抵押，利率会稍低，存款类机构间利率债质押回购利率，因为是一周的时间，所以叫DR007
    • 同业拆借利率高的时候，说明资金紧，银行倾向卖国债换取流动性，国债价格下降，不适合买入债券基金

第五节：央行的利率及其影响

  1. 银行缺钱会向央行借钱，因为利率最低（约等于国债利率）
  2. 银行间的借贷利率叫质押式回购利率DR，都缺钱的时候会上升
  3. 央行工具：
    • 每日国债逆回购
    • 每月中期借贷便利
  4. 利率影响
    • 影响债券：利率上升（别人更愿意去买新发售的债券，而不是从你这买），你手中的债券价格下跌
    • 影响股市：利率下调，利好股市
      • 公司支付的利息减少-成本降低
      • 刺激消费（不愿意存钱）
      • 贷款、投资意愿提升（存钱收益低）

第六节：国债

  1. 国债是金融体系的地基
  2. 财政部发布 - 金融机构作为一级承销商（大型银行、证券公司）- 商业银行、企业、居民
    • 国家获得了融资
    • 国债大多由金融机构持有
    • 由财政部到金融机构再到企业居民的国债发行途径，为一级市场，买卖“二手”国债的市场为二级市场
    • 央行不能在一级市场收购国债，但可以在二级市场操作，对经济进行宏观调控
  3. 国债作为融资工具，在经济发展中起到催化剂的作用
  4. 国债一开始是为了应对战争；国补是用国债的钱去刺激消费，增加税收
  5. 量化宽松：
    • 财政部发国债，由一级交易商（约 20 家，含投行、大行）在拍卖中买走。
    • 美联储（纽联储 SOMA 账户）在二级市场向这些一级交易商买国债。
    • 结算时：美联储给交易商在美联储账户上记一笔准备金（凭空创造），交易商把国债过户给美联储。
    • 交易商再把准备金转给自己或其他银行，银行体系准备金总额增加。
  6. 国家还不上国债了怎么办？
    • 国家信用体系崩溃，货币政策失效
      长期来看，物价水平总会调整到货币需求等于货币供给的水平

第七节：价格系统

  1. 价格长期会收敛到供给量 = 需求量的均衡价格。
  2. 做附加值 → 消费者觉得 “更值” → 愿意买的人变多 / 愿意花更多钱 → 需求变强 → 新均衡价更高
  3. CPI：居民消费价格指数Consumer Price Index
    • 老百姓买东西的物价（消费端物价），统计日常商品和服务的价格涨跌，衣食住行等等
    • CPI涨：代表物价涨，通货膨胀；钱不值钱，存款缩水，生活成本变高
    • 最好维持在2-3%
  4. PPI：工业生产者出厂价格指数
    • 工厂生产完卖给经销商的价格（生产端物价）
    • PPI涨：代表原料涨，工厂成本涨
  5. PPI涨 - 成本往下传 - CPI涨
  6. CPI高 - 通胀 - 央行容易收紧货币政策 - 利率上行 - 债基容易跌

第八节：股票（去看别的吧，不想写了）

  1. 对投资者此时此刻的预期的预测
  2. 确定自己的买卖逻辑，节约时间去无效看盘

第九节：货币总量

  1. 货币和贵金属挂钩时，总量是确定的；但和国家信用挂钩后
  2. M0：居民和企业手中的现金，流通中的货币
  3. M1: M0 + 数字资产（微信、支付宝、活期存款）
  4. M2：M1 + 定期存款、券商账户、公积金

第十节：如何衡量一个国家的经济水平？

  1. GDP计算方式
    • 生产法：每个环节多了的附加值 - 中国、亚洲
    • 支出法：消费者最后买了多少 - 美国日本
      • GDP = C + I + G + (X-M)
        • C：消费
        • I：投资
        • G：政府购买
        • X：出口总额
        • M：进口总额
    • 收入法：每个环节利润有多少 - 欧盟
      • GDP = 劳动者报酬（打工人）
        + 生产税净额（政府）
        + 固定资产折旧（企业）
        + 营业盈余（企业）
  2. 总量
    • 名义GDP：用今年的价格来算
    • 实际GDP：用去年的价格来算
    • 美元计价GDP：汇率影响
    • 不变价美元GDP
    • 购买力平价GDP
  3. 增速：同理总量
    • 偏差抵消，更具参考价值
  4. 人均数据、相对数据
    • 人均GDP：国民富裕程度
    • 生产效率：GDP/就业人数
    • 投资效率：投资总额/GDP增量
